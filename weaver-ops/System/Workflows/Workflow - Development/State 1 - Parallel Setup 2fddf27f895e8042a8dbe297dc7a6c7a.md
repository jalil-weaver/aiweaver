# State 1 - Parallel Setup

### **Goal**

Prepare all organizational, technical, and human prerequisites required to start execution in parallel without friction.

---

## **State Owner**

Project / Product Lead

---

## **Required Functions**

- Project / Product Lead
- Tech Lead
- Staffing / Capacity Manager

---

**Entry conditions**

- State 0 closed
- Functional documentation, technical documentation (including infra assumptions), and design validated

**Exit conditions**

- Work organization ready
- Delivery resources allocated or formally requested
- Repositories and tools ready
- Infrastructure and environments ready
- Required accesses and external accounts available

---

### **Actions**

**Action 1 – Prepare Work Organization**

R: Project / Product Lead · A: Project / Product Lead

Purpose: set up execution structure for delivery.

Must be true after: Kanban created; backlog structured; sprint or iteration cadence defined.

---

**Action 2 – Set Up Infrastructure and Environments**

R: Tech Lead · A: Tech Lead

Purpose: provision required infrastructure and environments for development.

Must be true after: environments deployed, accessible, and aligned with technical documentation where applicable.

---

**Action 3 – Allocate Delivery Resources**

R: Staffing / Capacity Manager · A: Staffing / Capacity Manager

Purpose: ensure that all required delivery functions are allocated through the dedicated workflow **“Allocate Delivery Resources.”**

Must be true after: delivery team allocated or formally requested; required functions locked to the project.

---

**Action 4 – Set Up Repositories and Tooling**

R: Tech Lead · A: Tech Lead

Purpose: prepare technical foundations for development.

Must be true after: repositories created; tooling configured; required accesses granted where applicable.

---

**Action 5 – Configure External Services and Accesses**

R: Tech Lead · A: Tech Lead

Purpose: ensure access to required external services and APIs.

Must be true after: required external accounts and API accesses available where applicable.
